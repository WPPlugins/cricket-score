== Description ==

Get the latest live <a href="http://toys.ocricket.com/cricket-score/" title="Cricket Live Score">Cricket Score</a> for your website.

== Installation ==

1. Upload folder `ocricket-score` to the `/wp-content/plugins/`
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Either use the Widget or Place `<?php oCricket_score_show(); ?>` in your templates
